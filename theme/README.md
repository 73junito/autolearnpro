Theme folder structure

- assets/
  - styles/  (CSS, SASS)
  - scripts/ (JS)
  - images/  (images)
  - fonts/   (webfonts)
- components/ (reusable UI components)
- templates/  (page templates)
- layouts/    (base layouts)
- static/     (static files served as-is)
- dist/       (build output)

Placeholders (.gitkeep) were added to ensure empty folders are tracked.
