# apps/web

Minimal Next.js (App Router) scaffold for the LMS frontend.

Run locally:

```bash
cd apps/web
npm install
npm run dev
```

Storybook:

```bash
cd apps/web
npm install
npm run storybook
```

Set env in `.env.local` (example in `.env.example`):

```
NEXT_PUBLIC_API_URL=http://localhost:4000
```
